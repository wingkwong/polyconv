Polyconv

A modular structured data format conversion toolkit for JSON, TOML, YAML, XML, CSV, TSV, INI, ENV, Markdown, HTML, and URL query string workflows.

Quick start View on npm
JSON YAML XML TOML CSV TSV INI ENV Markdown HTML Query

Quick start

Install the CLI or the focused packages you need.

Install

pnpm add -g @polyconv/cli
pnpm add @polyconv/json @polyconv/toml

Convert

polyconv convert input.json --to yaml --pretty
polyconv convert input.json --to xml --root-name package
polyconv convert input.toml --to json --output output.json

CLI

Convert, format, and minify from the terminal.

The CLI accepts files or stdin, infers JSON and TOML from filenames, and writes to stdout unless an output path is provided. 

polyconv convert data.json --to csv --output data.csv
polyconv convert data.json --to markdown --sort-keys
cat data.json | polyconv convert - --from json --to yaml
polyconv format data.json --indent 2 --sort-keys
polyconv minify data.json --output data.min.json

TypeScript

Use focused ESM packages in Node.js 22+ projects.

Install only the conversion surface you need. Packages ship TypeScript declarations and tree-shakeable exports. 

import {
  formatJson,
  jsonToCsv,
  jsonToQueryString,
  jsonToToml,
  jsonToXml,
  jsonToYaml,
} from "@polyconv/json";

const input = '{"name":"polyconv","version":"0.3.0"}';

const xml = jsonToXml(input, { rootName: "package", pretty: true });
const yaml = jsonToYaml(input, { pretty: true, sortKeys: true });
const toml = jsonToToml(input, { sortKeys: true });
const csv = jsonToCsv('[{"name":"polyconv","version":"0.3.0"}]');
const query = jsonToQueryString('{"q":"polyconv","tag":["json","cli"]}');
const formatted = formatJson(input, { indent: 2, sortKeys: true });
import { tomlToJson } from "@polyconv/toml";

const toml = 'name = "polyconv"\n\n[config]\nenabled = true\n';

const json = tomlToJson(toml, {
  pretty: true,
  indent: 2,
  sortKeys: true,
});
